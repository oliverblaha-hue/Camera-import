Camera Import
1. Pretahni aplikaci Camera Import do slozky Applications.
2. Ve Finderu otevri slozku Applications.
3. Klikni na Camera Import pravym tlacitkem a zvol Otevrit.
4. V zobrazenem okne znovu zvol Otevrit.
5. Pri prvnim spusteni povol oznameni, aby aplikace upozornila na pripojeny fotak.

Pokud macOS tlacitko Otevrit nenabidne:
1. Otevri Nastaveni systemu.
2. Prejdi do Soukromi a zabezpeceni.
3. U upozorneni na Camera Import zvol Presto otevrit.

Aplikace vyzaduje macOS 13 nebo novejsi.
